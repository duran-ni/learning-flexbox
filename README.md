# learning-flexbox
Proyecto sobre el aprendizaje de flexbox

---
# Descripción
Este proyecto es una interfaz web estática diseñada para presentar información técnica sobre HTML5 y CSS3. EL objetivo es implementar el concepto de flexbox,  que es un modelo de diseño cuya función es distribuir espacio y alinear elementos de forma eficiente dentro de un contenedor. Es ideal para crear estructuras web modernas y responsivas, facilitando la organización en filas o columnas.

# Análisis
El requerimiento principal consiste en visualizar dos bloques de contenido independientes. Cada bloque debe contener una representación visual de la tecnología y una descripción detallada. El desafío técnico reside en mantener la consistencia visual, el espaciado simétrico y la legibilidad de la tipografía personalizada.

# Estructura de la Página
La estructura se divide en tres niveles jerárquicos:

## Header: 
Contiene el título principal de la sección.

## Main Container: 
Una caja grande en el centro que guarda y alinea todas las tarjetas.

## Cards: 
Bloques individuales para cada tecnología con su imagen, título, descripción y enlace.

# Planificación
El desarrollo se dividió en las siguientes fases:

1. Estructura: Creación del archivo HTML base.  

2. Estilos: Definición de colores, sombras y bordes redondeados en CSS.  

3. Contenido: Inserción de textos descriptivos y logotipos.  

4. Ajustes: Refinamiento de márgenes y espacios internos.

# Diseño Visual
## Fondo:
Color gris claro para resaltar el contenido.

## Tarjetas: 
Fondo blanco con bordes redondeados y una sombra suave para dar efecto de elevación.

## Tipografía: 
Uso de varios tipos de fuentes.

# ♿ Accesibilidad
- Uso de etiquetas semánticas para mejorar la lectura por parte de asistentes de voz.

- Atributos `alt` en las imágenes para describir los logos de HTML5 y CSS3.

- Contraste de color adecuado entre el texto y el fondo de las tarjetas.
